# Complaint Tracker Web Application

A full-stack web application to register and track complaints.  
Built with **Node.js, Express.js, PostgreSQL** for the backend and **HTML + Tailwind CSS** for the frontend.

---

## 🚀 Features
- RESTful APIs for complaint registration and tracking
- PostgreSQL relational database for efficient storage and retrieval
- CRUD operations (Create, Read, Update, Delete)
- Simple frontend form with Tailwind styling
- Ready for authentication integration (JWT)

---
## 🛠 Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **Frontend:** HTML, Tailwind CSS
- **Tools:** Git, Postman
